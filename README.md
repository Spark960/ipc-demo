## Overview

The official submission dashboard for the Innovation Policy Consortium (IPC). This portal was developed to handle the submission of policy documents by participating colleges.

**Key Highlights:**
- **Adoption:** Successfully used by 50+ college teams.
- **Core Functionality:** Facilitates authenticated file uploads
- **Tech Stack:** Built with **Next.js** and **Supabase**, leveraging real-time data synchronization and secure file management with cookie based auth.


## To Run locally, clone and setup environment variables:
Then, run the development server:

```bash
npm i
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
